❓ Wie gebe ich einen Ordner von meinem Computer frei?

Browser dürfen aus Sicherheitsgründen keinen lokalen Pfad direkt an Hermes weitergeben. Du musst den Ordner als Netzwerk-Share bereitstellen, damit die VM ihn mounten kann. Dann hat er auf der VM einen Pfad wie /mnt/macshare, den du hier einträgst.

3 typische Wege:

  • SMB (macOS): Systemeinstellungen → Allgemein → Freigabe → Dateifreigabe.
    Auf VM: mount -t cifs //<mac-ip>/<share> /mnt/macshare -o user=…
  • sshfs: Mac hat sshd built-in. Auf VM:
    sshfs user@<mac-ip>:/Users/user/projekt /mnt/macprojekt
  • NFS-Export vom Mac, dann mount -t nfs <mac-ip>:/path /mnt/macnfs

Nach dem Mount: hier den VM-seitigen Pfad eintragen.
