{{ page_label }} sind noch nicht verfügbar

{{ page_label }} werden vom Hermes-Agent erzeugt — entweder per Cronjob (News, Briefing) oder beim Speichern einer Aufgabe in der App. Sobald der Agent die entsprechende Datei produziert hat, erscheint sie automatisch hier.

Häufige Ursachen

  • Hermes-Agent läuft noch nicht auf dem konfigurierten Host. Prüfe in Settings → 🛰️ App die Verbindungs-Einstellungen.
  • Connection-Mode = local, aber kein Hermes auf dem gleichen Host. Im HA-Add-on z. B. sinnvoll, wenn dein Hermes-Agent auf einer separaten VM läuft → in Settings auf ssh umstellen und IP/Key eintragen.
  • Pfad falsch: Portal sucht hier: {{ full_path }}
  • Cronjob ist noch nicht gelaufen. Du kannst den Briefing-Cron manuell anstoßen unter ☕ Briefing → „Jetzt erzeugen".
